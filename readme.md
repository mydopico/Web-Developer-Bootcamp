This repository contains the exercises and projects, including a gigantic production application called YelpCamp, for the course 'Web Developer Bootcamp' by Colt Steele on Udemy. 

This course covers:
* HTML5
* CSS3
* Bootstrap 
* SemanticUI
* DOM Manipulation
* jQuery
* Unix Commands
* NodeJS
* NPM
* ExpressJS
* API
* REST
* MongoDB
* Database Associations
* Authentication
* PassportJS
* Authorization
* Deployment
